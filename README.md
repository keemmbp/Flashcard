# Flashcard: อารยธรรมกรีกโบราณ

แฟลชการ์ดออนไลน์สำหรับทบทวนความรู้เกี่ยวกับอารยธรรมกรีกโบราณ  
สร้างด้วย HTML, CSS (TailwindCSS), และ JavaScript

## คุณสมบัติ

- แสดงคำถาม-คำตอบแบบแฟลชการ์ด
- พลิกการ์ดเพื่อดูคำตอบ
- เลือกดูการ์ดก่อนหน้า/ถัดไป
- สุ่มลำดับการ์ดใหม่โดยไม่ซ้ำ (Shuffle)
- แสดงสถานะลำดับการ์ด

## วิธีใช้งาน

1. เปิดไฟล์ `Flashcard.html` ด้วยเว็บเบราว์เซอร์
2. ใช้ปุ่มต่าง ๆ เพื่อควบคุมแฟลชการ์ด:
   - **ก่อนหน้า**: ดูการ์ดก่อนหน้า
   - **ถัดไป**: ดูการ์ดถัดไป
   - **พลิก**: พลิกการ์ดเพื่อดูคำตอบ
   - **สลับการ์ด**: สุ่มลำดับการ์ดใหม่

## การแก้ไข/เพิ่มข้อมูล

- แก้ไขหรือเพิ่มคำถาม-คำตอบได้ที่ตัวแปร `flashcardsData` ในไฟล์ `Flashcard.html`

## ข้อกำหนด

- ใช้งานได้ทันที ไม่ต้องติดตั้งอะไรเพิ่มเติม
- รองรับทั้งคอมพิวเตอร์และมือถือ
